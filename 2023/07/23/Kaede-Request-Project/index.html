银川

简称KRP，现在已经更新到了2.0版本，为了更好的可迁移性，我直接在这里展示整个项目

下面我就开始介绍这个东西的使用方法吧源代码分析我已经在源代码的注释里面写的很详细了，在这里我就不再说一遍哩

介绍

其实就是如题，一个用来请求的项目程序，我写这个代码的最初目的是用来下载涩图，反正就是因为使用习惯之类的原因，我逐渐构建了一个差不多的用户界面，然后优化了功能以及其他的一些小细节，最后就得到了一个这样子的项目~

我提供了几个模块，第一个模块当然就是下载模块啦，直接输入链接，以及下载次数，点击下载即可（链接应当能够直接跳转到一张图片，302跳转也可以）

第二个模块就是去重，把重复的内容删掉，这个是通过MD5检测实现的，和文件名称无关，所以不用担心

使用规范

不允许用来盈利，不允许用来违法用途，自己玩玩就足够了
