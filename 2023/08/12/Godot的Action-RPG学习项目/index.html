Godot的Action RPG学习项目

引入

嘛，这应该是我的第一篇关于Godot的博客吧？毕竟也是我最近开始学习的啦，倒也是正常~

实话说，我觉得Godot算是非常简单的一个游戏开发引擎了，比之前学的Unity和UE要简单不知道多少，尤其是这个引擎的大小！是真的小，小到我这种古老的电脑都能流畅运行；好啦不多说，直接开始吧~

介绍

本项目实现了一个基本的ARPG (Action RPG)游戏，使用中文进行注释，大部分代码都有注释，目前没有明显的Bug

项目使用环境：

Godot: Godot_v3.5.2-stable_win64

RAM: 8G

System: Win11

Code Language: GDScript

如果你要问我为什么不用最新版，原因就是我的电脑用不了…说不定以后我会找到解决方案吧，不过目前我没有什么好的方案呢
