解决new post报错的问题
发表于2023-07-23 技术 Hexo
在使用Hexo的时候，想要创建一篇新的文章，在控制台输入hexo new post “Title”的时候，可能会出现此电脑无法运行脚本的报错，特别是在Win11系统上。 原因在计算机上启动 Windows PowerShell 时，执行策略很可能是 Restricted。 Restricted 执行策略不允许任何脚本运行。 AllSigned 和 RemoteSigned 执行策略可防止 Windows PowerShell 运行没有数字签名的脚本。 解决方案可以通过如下命令查看策略组： get-executionpolicy 可以通过如下命令，修改策略组，从而解决问题： set-executionpolicy remotesigned
